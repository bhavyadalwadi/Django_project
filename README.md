# Django_project

Tutorial-driven Django practice repo that contains small exploratory projects, including a stock API example and a music-site scaffold.

## Why this repo exists

This repo captures early Django learning work rather than a polished product. It is useful as a record of backend framework practice, model/view structure, and local Django project setup.

## What is inside

- `Stock_site/`: Django project with a simple stock model and API-style view
- `Music_site/`: separate Django project scaffold
- Local SQLite databases for tutorial experimentation

## Run locally

1. Open one project folder such as `Stock_site/`
2. Confirm `manage.py` is present
3. Run:
   `python manage.py runserver`
4. Open the local Django URL shown in the terminal
